Weather forecast

  • Tech Stack: Django, Python, HTML, CSS , weather API
  • Github URL: Project Link

I am excited to share that I have successfully developed a weather forecast website project, showcasing my passion for web development and my commitment to providing users with accurate and accessible weather information. Through this project, I aimed to create a user-friendly platform that delivers real-time weather updates in a visually appealing and intuitive manner.

Utilizing a combination of front-end and back-end technologies, I designed the website to be responsive, ensuring a seamless experience across various devices. The user interface is crafted to be intuitive, allowing users to easily navigate and access detailed weather forecasts for their desired locations.
